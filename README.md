## 🚀 Projects Included

### 1. Interactive Calculator (Portfolio-Level)
A professional calculator featuring:

- Basic arithmetic: `+`, `-`, `*`, `/`  
- History of previous calculations  
- Dark/light theme support  
- Mobile responsive layout  
- Data persistence with `localStorage`  

**File:** `index.html`

### 2. Advanced Student Grade Tracker
A full-featured grade management tool:

- Add, edit, delete students  
- Auto-calculation of grades (A–F)  
- Real-time search/filter  
- Dashboard: Average, Total, Top performer  
- Activity log of last 5 actions  
- Data saved in `localStorage`  
- Fully responsive, centered layout  

**File:** `grade-tracker.html`
