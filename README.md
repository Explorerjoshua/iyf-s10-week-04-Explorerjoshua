## 🚀 Projects Included

### 1. 🧮 Interactive Calculator

A simple web-based calculator built using HTML and JavaScript that performs basic arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

It accepts user input and displays results instantly, providing a clean and interactive user experience.

---

### 🎓 Student Grade Tracker (Advanced)
An interactive web app that:
- Stores student names and marks using arrays & objects
- Automatically assigns grades
- Calculates class average
- Edit/Delete students
- Search/filter students
- Displays a grade distribution chart
- Saves data using localStorage
- Responsive and mobile-friendly

👉 Open `grade-tracker.html` to use the app
